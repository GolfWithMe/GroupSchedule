---Database----

1. Run SQL from folder db->sql.txt
2. You can modify database table prefix "akzwy" if require then run sql.

---Component----

1. Extract components.zip and upload all files under "html" folder.
2. Add menu from admin panel.
2.a. go to administrator -> Menus -> Manage -> JomSocial toolbar -> New
2.b. give name "Golf With Me" select ->Community->View Requests. 
2.c. Parent Item - Main Item Root -> Save

3. Add Sub Menu
3.a. go to administrator -> Menus -> Manage -> JomSocial toolbar -> New
3.b. give name "New Request" select ->Community->New Request. 
3.c. Parent Item - Golf With Me -> Save

4. Add Sub Menu
4.a. go to administrator -> Menus -> Manage -> JomSocial toolbar -> New
4.b. give name "View Requests" select ->Community->View Requests. 
4.c. Parent Item - Golf With Me -> Save

you can set Status, Access etc..


---Module----

1. go to administrator -> Extensions -> Manage -> Install -> Upload Package File
2. Browse file mod_community_groupschedule.zip -> Upload 
3. Add Modules
3.a. administrator -> Extensions -> Modules -> New -> Golf With Me 
3.b. select Position -> sidebar-2, Access - Registered, Order, Status update
3.c. Menu Assignment -> All pages -> Save
